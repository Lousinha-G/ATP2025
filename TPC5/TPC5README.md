## Resumo:
Este trabalho teve como objetivo criar um programa de gestão de turmas, utilizando listas de listas, incluindo funções de criação e abertura de ficheiros.

- aluno = (nome, id, [notaTPC, notaProj, notaTeste])

- turma = [aluno]

Para realizar a gestão da turma, utilizei as seguintes funções:
- criarTurma: cria uma turma;
- inserirAluno: insere um aluno na turma;
- listarTurma: lista todos os alunos da turma;
- consultarAluno: consulta um aluno por id;
- calcularMediaAluno: calcula a média de um aluno;
- calcularMediaTurma: calcula a média da turma;
- guardarficheiro: guarda a turma num ficheiro;
- abrirficheiro: abre uma turma a partir de um ficheiro;

No programa principal, implementei um menu que permite ao utilizador escolher entre as diferentes funcionalidades.
