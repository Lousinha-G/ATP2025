## Resumo:
Este trabalho teve como objetivo criar um programa de gestão de cinemas, utilizando listas.
Utilizando os seguintes objetos:

- Cinema = [Sala]
- Sala = [nlugares, Vendidos, filme]
- nlugares = Int
- filme = String 
- Vendidos = [Int]

Para realizar a gestão dos objetos, utilizei as seguintes funções:
- listar: lista todos os filmes em exibição;
- disponível: verifica se um certo lugar numa sala está disponível;
- vendebilhete: altera o estado de um certo lugar numa sala para ocupado;
- inserirsala: pede as informações de uma sala, e insere-a na lista Cinema;
